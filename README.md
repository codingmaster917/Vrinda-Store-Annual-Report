# Vrinda Store Annual Report Dashboard

## Project Overview
This is an Excel Dashboard created using the Vrinda Store sales dataset.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

## Dashboard Features
- Orders vs Sales Analysis
- Sales: Men vs Women
- Order Status Analysis
- Top 5 States by Sales
- Age vs Gender Analysis
- Orders by Sales Channel

## Dashboard Preview
## Key Insights
- Women contribute around 64% of total sales.
- Delivered orders are approximately 92%.
- Maharashtra has the highest sales.
- Amazon contributes the highest number of orders.
